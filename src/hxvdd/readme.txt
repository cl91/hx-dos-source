
 About HXVDD

 This VDD installs support for the WinNT clipboard for HX DOS apps.

 Installation

 HXVDD.DLL must be copied to a directory included in the PATH
 environment variable. AFAIK, for Vista, it has to be copied to the
 Windows system directory.
